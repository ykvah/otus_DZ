# Корневой коммутатор - S1

Причина - Bridge ID равны а у S1 наименьший MAC

# Корневые порты 

S2 - Et0/2

# Назначенные порты

S1 - Et0/0 Et0/2
S2 - Et0/0

# Альтернативный порт 

S3 - Et0/2 - Единственный альтернативный путь к Root 
+ у S3 Bridge ID меньше поэтому порт Et0/0 на S2 - назначенный
